3.7 Relaxation to Linear Programming 对线性规划的松弛

3.7.1 基本概念
Linear Programming 线性规划(LP)

在《算法导论》中学习过线性规划的基本定义

形式化地描述如下：是一个线性函数最小化或最大化的问题，该线性函数服从一组有限个线性约束。

0/1-LP、IP：integer programming

区别在于解被限制在集合${0,1}$、整数上，问题的难度提升了，属于NP-hard问题

线性规划的松弛

step1: Reduction规约

将优化问题U的输入实例转化成0/1-LP or IP的输入实例

step2: Relaxation松弛

利用一般的线性规划的算法计算出最优解$\alpha$，放宽了解空间的规定

step3: Solving the original problem

$cost(\alpha)$是原来问题的解的上限，利用$\alpha$计算原问题的最优解或者是更高质量的近似解

前两步可以保证在多项式时间内而完成，第三步的时间复杂度根据问题的难度和对于解的要求决定。

3.7.2 将问题表述为线性规划问题
线性规划问题的形式

standard（equality）form

$线性函数：c^{T}·X=\sum_{i=1}^{n}c_{i}·x_{i}$

$约束条件1：A·X=b,i.e.,\sum\limits_{i=1}^{n}a_{ji}x_{i}=b_{j}\ for\ j=1..m$

$约束条件2：x_{i}\geq 0\ for\ i=1..n$

对于IP而言，需要系数也为整数

canonical form

$线性函数：c^{T}·X=\sum_{i=1}^{n}c_{i}·x_{i}$

$约束条件1：A·X\geq b,i.e.,\sum\limits_{i=1}^{n}a_{ji}x_{i}\geq b_{j}\ for\ j=1..m$

$约束条件2：x_{i}\geq 0\ for\ i=1..n$

standard inequality form

$线性函数：c^{T}·X=\sum_{i=1}^{n}c_{i}·x_{i}$

$约束条件1：A·X\leq b,i.e.,\sum\limits_{i=1}^{n}a_{ji}x_{i}\leq b_{j}\ for\ j=1..m$

$约束条件2：x_{i}\geq 0\ for\ i=1..n$

将canonical形式转换为standard（equality）形式

引入$surplus\ variable$

由

$\sum\limits_{i=1}^{n}a_{ji}x_{i}\geq b_{j}\ $

得到

$\sum\limits_{i=1}^{n}a_{ji}x_{i}-s_{j}=b_{j},s_{j}\geq0$

原来的可行解为：

$(x_{1},x_{2},\dots,x_{n})$

现在的可行解为：

$(x_{1},x_{2},\dots,x_{n},s_{1},s_{2},\dots,s_{m})$

转化standard inequality form时同理。

将最大化问题转化为最小化问题

$目标：maximize\ c^{T}·X=\sum\limits_{i=1}^{n}c_{i}x_{i}$

等价于

$minimize\ [(-1)·c^{T}]·X=\sum\limits_{i=1}^{n}(-c_{i})·x_{i}$

SET COVER PROBLEM 集合覆盖问题SCP

布尔变量$x_{i}$含义是是否选择集合$S_{i}$，其中$S_{i}\subseteq X={a_{1},a_{2},\dots,a_{n}}$

目标函数：

$minimize\ \sum\limits_{i=1}^{m}x_{i}$

约束条件：

$\sum\limits_{j\in Index(k)}x_{j}\geq 1\ for\ k=1,\dots,n.$

$Index(k)={d\in{1,\dots,m}|a_{k}\in S_{d}}$

$a_{k}$所在的集合的下标的集合，约束条件即为每个$x_{j}$被选到至少一次，即为覆盖了整个$X$集合

Minimum Vertex Cover Problem（MIN-VCP）最小点覆盖问题
MINIMUM WEIGHTED VERTEX COVER最小带权点覆盖问题

目标：

$minimize\ \sum\limits_{i=1}^{n}c(v_{i})·x_{i}$

约束条件：

$x_{i}+x_{j}\geq 1\ for\ every\ {v_{i},v_{j}}\in E$

Makespan Scheduling Problem（MS） 最小工期调度问题

输入：$p_{1},p_{2},\dots,p_{n}$时间分配到m台相同的机器上

目标：使得完成所有任务的用时最短

线性规划形式：

$x_{ij}\in {0,1}\ for\ i=1,\dots,n,j=1,\dots,m第i个任务分配到第j台机器上$

$\sum\limits_{j=1}^{m}x_{ij}=1\ for\ all\ i\in{1,\dots,n}保证每个任务都在一台机器上执行过一次$

$minimize\ t$

$t-\sum\limits_{i=1}^{n}p_{i}·x_{ij}\geq 0\ for\ all\ j\in{1,\dots,m}$

优化问题可以自然地规约到0/1-LP问题

可行解一一对应

$M_{U}(x)=M_{IP}(y)$，且最优解也同时对应

3.7.4 Rounding, LP-Duality and Primal-Dual Method
Rounding方法

考虑集合覆盖问题的子问题SCP（k）

每个元素出现的集合个数最多为k个

即$\mid Index(a_{j})\mid\leq k$

Algorithm

step1:Reduction
step2:Relaxation
解空间放宽到$0\leq x_{i}\leq 1$

step3:Solving the original problem

​ $Set\ \beta_{i}=1\ iff\ \alpha_{i}\geq \frac{1}{k}$

output:$\beta=(\beta_{1},\dots,\beta_{m})$

如何保证得到的是可行解？它是最优解吗？
