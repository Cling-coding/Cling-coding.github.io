Go和Goroutine

Goroutine
  • 每个 CPU 上有一个 Go Worker（线程），自由调度 goroutines（协程）
  • 执行到 blocking API 时 (例如 sleep, read)
    • Go Worker 偷偷改成 non-blocking 的版本
      • 成功 → 立即继续执行
      • 失败 → 立即 yield 到另一个需要 CPU 的 goroutine
